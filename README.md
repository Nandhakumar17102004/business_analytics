# Bank Marketing Analytics Project

## Overview
This project focuses on analyzing bank marketing campaigns to understand customer behavior and improve marketing strategies. The analysis provides insights into customer segments, campaign performance, and recommendations for future campaigns using Power BI.

## Objectives
- To explore and analyze historical bank marketing data.
- To identify patterns and trends in customer responses.
- To evaluate the effectiveness of different marketing strategies.
- To provide actionable insights for optimizing future campaigns.

## Dataset
The dataset used in this project consists of various features related to customer demographics, marketing campaigns, and their outcomes. Key features include:
- `age`: Age of the customer.
- `job`: Type of job.
- `marital`: Marital status.
- `education`: Level of education.
- `balance`: Customer's bank balance.
- `contact`: Type of communication (e.g., phone).
- `duration`: Duration of the last contact in seconds.
- `y`: Target variable, indicating if the customer subscribed to a term deposit.

## Methodology
1. **Data Cleaning**: Handle missing values and outliers.
2. **Exploratory Data Analysis (EDA)**: Visualize data points using graphs and plots to find insights.
3. **Modeling**: Apply various machine learning algorithms to predict customer response.
4. **Evaluation**: Assess model performance using metrics such as accuracy, precision, and recall.
